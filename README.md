# EmotiCare AI 🧠💬

An AI-powered emotion-aware chatbot that interacts with users based on their feelings and provides supportive responses.

## 🚀 Features

* Emotion detection (sad, happy, angry, stress, neutral)
* Context-based responses
* Chat history memory
* Interactive UI using Streamlit

## 🛠️ Tech Stack

* Python
* Streamlit

## ▶️ How to Run

```bash
pip install streamlit
python -m streamlit run app.py
```

## 📌 Project Purpose

This project demonstrates basic NLP logic and conversational AI design, focusing on user emotion understanding and interaction.
